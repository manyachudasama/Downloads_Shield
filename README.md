# Downloads_Shield
Downloads Shield is a Chrome extension designed to protect users from dangerous and suspicious file downloads. It monitors all downloads in real time, automatically blocking risky file types such as `.exe`, `.js`, or disguised files like `file.pdf.exe`, and warns users about potentially harmful files such as ZIP archives. The extension analyzes filenames, MIME types, and suspicious keywords, logs all actions, and provides a warning popup that allows users to confirm or cancel downloads. With a simple popup interface, users can toggle protection on or off, ensuring smart, proactive download security without disrupting regular browsing.
